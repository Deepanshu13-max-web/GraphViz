What is BFS Algorithm?

Breadth-First Search (BFS) ek graph traversal algorithm hai jo graph ya tree ke nodes ko level by level explore karta hai. Iska matlab pehle current node ke saare neighbors visit hote hain, phir unke neighbors, aur aise hi aage.

🎯 Key Points:

  • Queue data structure use hota hai (FIFO)
  • Level order traversal karta hai
  • Shortest path find karta hai (unweighted graphs me)
  • Connected components dhundh sakte hain

🌍 Real Life Examples:

GPS Navigation

Jab aap Google Maps me shortest route dhundhte ho (bina weight ke), to BFS ka concept use hota hai. Ye pehle nearby roads check karta hai, phir unse connected roads.

Social Networks

LinkedIn me "2nd degree connections" dikhana - BFS se pata chalata hai ki do log kitne levels me connected hain.

Web Crawlers

Google jaise search engines pehle current page ke links visit karte hain (BFS style).

How BFS Algorithm Works?

1

Initialize

Ek queue lo, ek visited array lo. Start node ko queue me daalo aur visited mark karo.

queue = [A]
visited = {A}
2

Process Node

Queue se ek node nikaalo (dequeue). Use process karo.

current = A
print(A)
3

Add Neighbors

Current node ke saare unvisited neighbors ko queue me daalo aur visited mark karo.

neighbors = [B, C]
queue = [B, C]
4

Repeat

Jab tak queue empty na ho, step 2-3 repeat karo.

while queue:
  process()

Pseudocode

def bfs(graph, start_node):
    # Create queue and visited set
    queue = [start_node]
    visited = set([start_node])

    while queue:
        # Dequeue a vertex
        current = queue.pop(0)
        print(current, end=" ")

        # Get all adjacent vertices
        for neighbor in graph[current]:
            if neighbor not in visited:
                visited.add(neighbor)
                queue.append(neighbor)
                    
void bfs(Graph graph, Node start) {
    Queue<Node> queue = new LinkedList<>();
    Set<Node> visited = new HashSet<>();

    queue.add(start);
    visited.add(start);

    while (!queue.isEmpty()) {
        Node current = queue.poll();
        System.out.print(current + " ");

        for (Node neighbor : graph.getNeighbors(current)) {
            if (!visited.contains(neighbor)) {
                visited.add(neighbor);
                queue.add(neighbor);
            }
        }
    }
}

Complexity Analysis

Case Time Complexity Space Complexity Explanation
Best Case O(1) O(1) Target immediately found (if searching)
Average Case O(V + E) O(V) Visit all vertices and edges once
Worst Case O(V + E) O(V) Complete graph, all nodes connected

🔍 Detailed Analysis:

  • V (Vertices): Har vertex ek baar queue me aata hai - O(V)
  • E (Edges): Har edge do baar check hoti hai (directed me ek baar) - O(E)
  • Space: Queue me at most V nodes store hote hain - O(V)

Note: BFS ka time complexity O(V + E) hota hai kyunki har vertex aur edge ek baar process hoti hai.

class Graph:
    def __init__(self):
        self.graph = {}

    def add_edge(self, u, v):
        if u not in self.graph:
            self.graph[u] = []
        if v not in self.graph:
            self.graph[v] = []
        self.graph[u].append(v)
        self.graph[v].append(u)  # For undirected graph

    def bfs(self, start):
        visited = set()
        queue = [start]
        visited.add(start)

        while queue:
            vertex = queue.pop(0)
            print(vertex, end=' ')

            for neighbor in self.graph[vertex]:
                if neighbor not in visited:
                    visited.add(neighbor)
                    queue.append(neighbor)

# Example usage
g = Graph()
g.add_edge('A', 'B')
g.add_edge('A', 'C')
g.add_edge('B', 'D')
g.add_edge('C', 'E')
print("BFS Traversal starting from A:")
g.bfs('A')  # Output: A B C D E
                    
class GraphMatrix:
    def __init__(self, size):
        self.size = size
        self.matrix = [[0] * size for _ in range(size)]

    def add_edge(self, u, v):
        self.matrix[u][v] = 1
        self.matrix[v][u] = 1  # For undirected graph

    def bfs(self, start):
        visited = [False] * self.size
        queue = [start]
        visited[start] = True

        while queue:
            vertex = queue.pop(0)
            print(vertex, end=' ')

            for i in range(self.size):
                if self.matrix[vertex][i] == 1 and not visited[i]:
                    visited[i] = True
                    queue.append(i)

# Example usage
g = GraphMatrix(5)
g.add_edge(0, 1)  # A-B
g.add_edge(0, 2)  # A-C
g.add_edge(1, 3)  # B-D
g.add_edge(2, 4)  # C-E
print("BFS Traversal starting from 0 (A):")
g.bfs(0)  # Output: 0 1 2 3 4
                    
class Node:
    def __init__(self, value):
        self.value = value
        self.neighbors = []

    def add_neighbor(self, node):
        self.neighbors.append(node)

class GraphOO:
    def __init__(self):
        self.nodes = {}

    def add_node(self, value):
        if value not in self.nodes:
            self.nodes[value] = Node(value)

    def add_edge(self, u, v):
        self.add_node(u)
        self.add_node(v)
        self.nodes[u].add_neighbor(self.nodes[v])
        self.nodes[v].add_neighbor(self.nodes[u])

    def bfs(self, start_value):
        if start_value not in self.nodes:
            return

        start = self.nodes[start_value]
        visited = set()
        queue = [start]
        visited.add(start)

        while queue:
            current = queue.pop(0)
            print(current.value, end=' ')

            for neighbor in current.neighbors:
                if neighbor not in visited:
                    visited.add(neighbor)
                    queue.append(neighbor)

# Example usage
g = GraphOO()

g.add_edge('A', 'C')
g.add_edge('B', 'D')
g.add_edge('C', 'E')
print("BFS Traversal starting from A:")
g.bfs('A')  # Output: A B C D E

Frequently Asked Questions

❓ BFS vs DFS - kya difference hai?

BFS queue use karta hai (level by level), DFS stack use karta hai (depth first). BFS shortest path deta hai unweighted graphs me, DFS cycle detection me useful hai.

❓ BFS ka time complexity O(V+E) kyun hai?

Kyunki har vertex ek baar process hota hai (V operations) aur har edge do baar check hoti hai (E operations). Total V + E.

❓ BFS weighted graph me shortest path dega?

Nahi, BFS sirf unweighted graphs me shortest path deta hai. Weighted graphs ke liye Dijkstra use karte hain.

❓ BFS ka space complexity O(V) kyun hai?

Queue me worst case me saare nodes aa sakte hain (V) aur visited array me bhi V nodes store hote hain.
