🔍 What is DFS Algorithm?

Depth-First Search (DFS) ek graph traversal algorithm hai jo graph ko depth me explore karta hai. Ye stack data structure use karta hai (recursion ya explicit stack).

🎯 Key Points:

  • Stack data structure use hota hai (LIFO)
  • Depth wise traversal karta hai
  • Recursive ya iterative dono tarike se implement kar sakte
  • Cycle detection aur topological sort me useful

🌍 Real Life Examples:

Maze Solving

Bhai jaise maze solve karte hain - pehle ek raaste depth me jao, agar rasta band ho to wapas aao.

⚙️ How DFS Works?

1

Start

Start node ko stack me daalo

stack = [A]
2

Pop & Visit

Stack se node nikaalo, visit karo

current = A (visited)
3

Push Neighbors

Unvisited neighbors ko stack me daalo

push B, C → stack = [B, C]
4

Repeat

Jab tak stack empty na ho

next = C (go deeper)
    A
   / \
  B   C
 / \   \
D   E   F

DFS Traversal: A → C → F → B → E → D

📝 Pseudocode

def dfs_recursive(graph, node, visited=None):
    if visited is None:
        visited = set()

    visited.add(node)
    print(node, end=" ")

    for neighbor in graph[node]:
        if neighbor not in visited:
            dfs_recursive(graph, neighbor, visited)

def dfs_iterative(graph, start):
    visited = set()
    stack = [start]

    while stack:
        node = stack.pop()
        if node not in visited:
            print(node, end=" ")
            visited.add(node)

            # Push neighbors in reverse order for correct order
            for neighbor in reversed(graph[node]):
                if neighbor not in visited:
                    stack.append(neighbor)
