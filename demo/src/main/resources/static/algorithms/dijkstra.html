Dijkstra's Algorithm

Shortest path algorithm for weighted graphs

Time: O((V + E) log V) Space: O(V) Type: Shortest Path

🎯 What is Dijkstra's Algorithm?

Dijkstra's algorithm weighted graph me shortest path find karta hai ek source node se sabhi nodes tak.

Key Points:

  • Priority Queue (Min Heap) use hota hai
  • Greedy approach follow karta hai
  • Non-negative weights required
  • GPS, Maps me use hota hai

📊 Step by Step Example

    A --4-- B
    |       |
    2       1
    |       |
    C --3-- D

Shortest path from A to D: A → C → D (Total: 5)
