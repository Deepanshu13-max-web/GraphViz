Kruskal's Algorithm

Minimum Spanning Tree - Edge based approach with Union-Find

Time: O(E log E) Type: MST

🔗 What is Kruskal's Algorithm?

Kruskal's algorithm bhi Minimum Spanning Tree banata hai, lekin ye edge-based approach use karta hai. Union-Find data structure se cycles detect karta hai.

🎯 Key Points:

  • Edges ko weight ke hisaab se sort karo
  • Union-Find se cycle detection
  • Smallest edge jo cycle na banaye, use MST me add karo
  • Sparse graphs ke liye better hai

📝 Pseudocode with Union-Find

class UnionFind:
    def __init__(self, n):
        self.parent = list(range(n))
        self.rank = [0] * n

    def find(self, x):
        if self.parent[x] != x:
            self.parent[x] = self.find(self.parent[x])
        return self.parent[x]

    def union(self, x, y):
        px, py = self.find(x), self.find(y)
        if px == py:
            return False

        if self.rank[px] < self.rank[py]:
            self.parent[px] = py
        elif self.rank[px] > self.rank[py]:
            self.parent[py] = px
        else:
            self.parent[py] = px
            self.rank[px] += 1
        return True

def kruskal(vertices, edges):
    # edges = [(weight, u, v), ...]
    edges.sort()  # Sort by weight
    uf = UnionFind(vertices)
    mst = []
    total_weight = 0

    for weight, u, v in edges:
        if uf.union(u, v):
            mst.append((u, v, weight))
            total_weight += weight

    return mst, total_weight
