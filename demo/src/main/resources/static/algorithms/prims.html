Prim's Algorithm

Minimum Spanning Tree - Vertex based approach

Time: O(E log V) Type: MST

🌳 What is Prim's Algorithm?

Prim's algorithm ek Minimum Spanning Tree (MST) banata hai graph ka. Ye vertex-based greedy algorithm hai.

1

Start

Kisi bhi vertex se start karo

2

Find Min Edge

Visited aur unvisited ke beech minimum weight edge dhundho

3

Add to MST

Us edge ko MST me add karo aur naya vertex visit karo

4

Repeat

Jab tak V-1 edges na mil jayein

📝 Pseudocode

PrimMST(graph):
    visited = empty set
    mst = empty set
    pq = priority queue

    start = any vertex
    visited.add(start)
    add all edges of start to pq

    while pq is not empty and visited.size < total_vertices:
        minEdge = pq.extractMin()

        if both ends visited:
            continue

        newVertex = unvisited end of minEdge
        visited.add(newVertex)
        mst.add(minEdge)

        for each edge of newVertex:
            if other end not visited:
                pq.add(edge)

    return mst
