What is a Graph?

Graph ek non-linear data structure hai jo nodes (vertices) aur edges se milkar banta hai. Jaise social network me log (nodes) aur unki friendship (edges).

🌍 Real Life Examples:

Social Networks

Nodes: Users | Edges: Friendships

Maps & Navigation

Nodes: Cities | Edges: Roads

Internet

Nodes: Websites | Edges: Hyperlinks

Graph Terminology

Term Meaning Example
Vertex/Node Graph ka basic unit (point) City, Person, Website
Edge Nodes ke beech connection Road, Friendship, Link
Path Nodes ka sequence A → B → C
Cycle Path jisme start aur end same A → B → C → A
Degree Node se kitne edges connected Node A ki degree 3

Types of Graphs

🔄

Directed Graph

Edges ka direction hota hai

A → B (A se B ja sakte)
🔗

Undirected Graph

Edges bidirectional hote hain

A — B (Dono taraf ja sakte)
⚖️

Weighted Graph

Har edge ka weight/cost hota hai

A -(5)- B (Distance 5)
1️⃣

Unweighted Graph

Sab edges ka weight same (1)

A — B (Weight = 1)
