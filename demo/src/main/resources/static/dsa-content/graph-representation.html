📝 Graph Representation

Graph ko represent karne ke do main tarike hain:

1. Adjacency Matrix

     A  B  C  D
  A [0, 1, 1, 0]
  B [1, 0, 0, 1]
  C [1, 0, 0, 1]
  D [0, 1, 1, 0]
  • Space: O(V²)
  • Edge check: O(1)
  • Find neighbors: O(V)
  • Best for: Dense graphs

2. Adjacency List

A → [B, C]
B → [A, D]
C → [A, D]
D → [B, C]
  • Space: O(V + E)
  • Edge check: O(degree)
  • Find neighbors: O(1)
  • Best for: Sparse graphs

💻 Implementation

class Graph:
    def __init__(self):
        self.graph = {}

    def add_edge(self, u, v):
        if u not in self.graph:
            self.graph[u] = []
        if v not in self.graph:
            self.graph[v] = []
        self.graph[u].append(v)
        self.graph[v].append(u)  # For undirected
