📊 Types of Graphs

1. Directed Graph (Digraph)

A → B (A se B, but B se A nahi)

Edges ka direction hota hai. Jaise Twitter follows, one-way roads.

2. Undirected Graph

A — B (Dono taraf)

Edges bidirectional hote hain. Jaise Facebook friendship, two-way roads.

3. Weighted Graph

A -(5)- B

Har edge ka weight/cost hota hai. Jaise distance, time, price.

4. Unweighted Graph

A — B

Sab edges ka weight same (usually 1) maana jata hai.

5. Cyclic Graph

A → B → C → A

Graph me kam se kam ek cycle ho.

6. Acyclic Graph

A → B → C

Koi cycle na ho. Jaise trees, DAG.

7. Connected Graph

Every node reachable

Har node se har node tak path ho.

8. Disconnected Graph

Multiple components

Kuch nodes reachable nahi hain.

📌 Important:

Ek graph multiple types ka ho sakta hai - jaise Directed Weighted Cyclic Graph, Undirected Unweighted Acyclic Graph (Tree), etc.
